# 线性回归背后的数学原理详解

> 写给数学基础薄弱的读者 — 从直觉出发，逐步深入。

---

## 一、前置概念：导数到底在说什么？

### 1.1 导数的直观理解

想象你开车从 A 城到 B 城，你的**位置**随时间在变化。导数是**瞬时速度**。

换个说法：导数描述的是**“变化趋势”**。

- 导数为**正**：函数值在**增大**（上坡）
- 导数为**负**：函数值在**减小**（下坡）
- 导数为**0**：函数值瞬间**不变**（站在平地上，可能是山顶或山谷）

### 1.2 用一个简单的抛物线来感受

拿最简单的二次函数来说：

```
f(x) = x²
```

它的导数是：

```
f'(x) = 2x
```

| x 的值 | 导数 f'(x) | 函数值 f(x) | 说明             |
| ---- | -------- | -------- | -------------- |
| -3   | -6       | 9        | 负导数，往下走        |
| -1   | -2       | 1        | 仍在下坡           |
| 0    | 0        | 0        | **导数为 0，停在谷底** |
| 1    | 2        | 1        | 正导数，往上走        |
| 3    | 6        | 9        | 仍在爬坡           |

关键观察：**在谷底 x=0 处，导数恰好为 0**。

这背后的直觉是：你站在谷底的时候，无论往左滑还是往右滑，都会往上走——谷底就是那个唯一的“平点”。

## 二、为什么导数为 0 就能找到最小值？

### 2.1 极值点的必要条件

如果一个函数是**光滑的**（没有尖角、没有断崖），那么：

> **在最小值（或最大值）的位置，切线一定是水平的，即导数 = 0。**

想象你把一颗玻璃珠放在一个光滑的碗里：

- 玻璃珠最终会停在碗的**最低点**
- 在最低点，碗壁是**平的**（切线水平）
- 切线水平 = 导数等于 0

这就是“令导数等于 0 去求最小值”的几何直觉。

### 2.2 但导数等于 0 就一定是最小值吗？

**不一定。** 导数等于 0 可能是三种情况：

```
      🏔️ 极大值（山顶）        🏔️
     /    \                  /    \
    /      \     ═════      /      \
   /        \               /        \
──/──────────\──         ──/──────────\──
               \           /
                \         /
                 \       /
                  🕳️ 极小值（山谷）
```

1. **极小值**（山谷）：我们想要的
2. **极大值**（山顶）：导数也是 0，但是最大值
3. **鞍点**（像马鞍一样，一个方向是谷，另一个方向是脊）

### 2.3 那为什么最小二乘法可以用导数为 0 来求最小值？

因为**损失函数是凸函数**。

**凸函数**有一个“无敌”的性质：它只有**一个**谷底，没有山顶，没有马鞍。

```
凸函数长这样（碗的形状）：

   \              /
    \            /
     \          /
      \________/
       只有一个最低点
```

对于凸函数：

- 导数 = 0 的点**一定**是全局最小值（唯一的谷底）
- 不会出现“导数 = 0 但是个山顶”的尴尬

平方损失函数 `loss = (f(x) - y)²` 本身就是凸函数，多个平方和加起来还是凸函数，所以我们可以放心地让导数等于 0 去求最小值。

### 2.4 最小二乘法的推导（简化版）

以最简单的形式来说明。假设我们想用一条直线 `y = wx + b` 来拟合数据 `(x₁, y₁), (x₂, y₂), ..., (xₘ, yₘ)`。

损失函数（均方误差）：

```
            1    m
L(w, b) = ———   Σ  (wxᵢ + b - yᵢ)²
           2m   i=1
```

要求最小值，分别对 w 和 b 求偏导并令其为 0：

```
∂L/∂w = 0
∂L/∂b = 0
```

解这个方程组就得到了最小二乘法的矩阵形式：

```
W = (XᵀX)⁻¹ XᵀY
```

## 三、梯度下降：为什么一步步走能找到最小值？

### 3.1 梯度是什么？

**梯度的方向 = 函数上升最快的方向。**

就像你站在一座山上：

- 你环顾四周，找到**最陡的上坡方向** → 那就是梯度的方向
- 如果你想下山（减小函数值） → 你就该往梯度的**反方向**走

> **负梯度方向 = 函数下降最快的方向。**

### 3.2 用爬山类比理解梯度下降

```
你蒙着眼睛站在一座山上，想要走到山谷：

1. 用脚感受脚下的坡度（计算当前位置的梯度）
2. 朝最陡的下坡方向迈一小步（沿负梯度方向更新参数）
3. 重复直到脚下变平（梯度 ≈ 0，到达谷底）
```

每一步都选当前最能让你下降的方向走，最终就会到达谷底。

### 3.3 数学形式

对于参数 w 和 b，每一步更新：

```
w_new = w_old - α × (∂L/∂w)
b_new = b_old - α × (∂L/∂b)
```

其中 `α` 是**学习率**（步长），控制每一步走多大。

### 3.4 为什么这个方法一定能收敛到最小值？

这得益于损失函数的**凸性**：

1. **只有一个谷底**：凸函数不会存在“走到了一个局部谷底但不是全局最低”的问题
2. **每一步都在下降**：只要梯度不为 0，沿着负梯度方向走一小步，函数值一定减小
3. **梯度越来越小**：越接近谷底，坡度越缓，梯度越接近 0，步伐自然变小，不会跳过最小值

用更数学的语言：

> 对于凸函数，梯度下降的每一步都满足 `L(新参数) < L(旧参数)`（只要步长合适），而且由于函数有下界（最小值），这个递减的过程必然收敛。

### 3.5 学习率的影响

```
学习率太小：                 学习率太大：
下山太慢                   震荡甚至发散
山谷                       山谷
 \                          \    /\    /
  \                          \  /  \  /
   \    🐢                    \/    \/
    \                              /
     \___                         /
                                  🐇 跳过谷底，来回震荡
```

## 四、最小二乘法 vs 梯度下降 — 直观对比

|     | 最小二乘法               | 梯度下降           |
| --- | ------------------- | -------------- |
| 思路  | 直接解导数为 0 的方程，一步到位   | 沿着梯度一步步迭代靠近    |
| 类比  | 用公式直接算出山谷的精确坐标      | 蒙着眼睛一步步摸下山     |
| 优点  | 有解析解时一步得出精确答案       | 数据量大、特征多时也能用   |
| 缺点  | XᵀX 不可逆时无解；数据量大时计算慢 | 需要调学习率；只能得到近似解 |

## 五、核心总结

> **为什么导数为 0 能求最小值？**
> 
> 因为光滑函数在谷底处切线是水平的（导数 = 0）。而平方损失函数是凸函数，只有一个谷底，所以导数 = 0 的点一定是全局最小点。

> **为什么梯度下降能找到最小值？**
> 
> 梯度的方向是函数上升最快的方向，往反方向走就是在以最快速度下山。对于凸函数，每一步都在下降，最终必然到达唯一的谷底。

> **两者本质是同一件事：**
> 
> 最小二乘法是**直接解**”导数 = 0”；梯度下降是**迭代地沿着**导数的反方向走，直到导数 ≈ 0。

## 六、凸函数与凹函数

### 6.1 用”碗”和”山”来建立直觉

这是你能记住的最简单的定义：

> **凸函数** = 碗形 🥣（开口朝上，能接住雨水）
> 
> **凹函数** = 山形 ⛰️（开口朝下，雨水会滑落）

```
凸函数（碗）                    凹函数（山）
   \          /                  /\          /\
    \        /                  /  \        /  \
     \______/                  /    \______/    \
     能装水                       水会流走
```

这就是最直观的区分方式。

### 6.2 严格的数学定义

数学上有两个等价的角度来定义凸函数，你可以挑一个更容易理解的：

#### 角度一：弦在函数上方（几何定义）

在函数曲线上任取两点 A 和 B，画一条线段连起来：

```
凸函数：弦在曲线上方              凹函数：弦在曲线下方

    B                               A●——————●B
   /|    ●———● 这是弦               /        \
  / |      (在曲线上方)             /          \
 /  |                             /            \
A   |                           凸函数反过来

弦 ≥ 函数值                    弦 ≤ 函数值
```

> **凸函数**：连接曲线上任意两点的线段（弦），始终在函数图像的**上方或重合**。
> 
> **凹函数**：连接曲线上任意两点的线段（弦），始终在函数图像的**下方或重合**。

用公式表达（这就是 Jensen 不等式）：

```
凸函数：f(λa + (1-λ)b) ≤ λ·f(a) + (1-λ)·f(b)
凹函数：f(λa + (1-λ)b) ≥ λ·f(a) + (1-λ)·f(b)

其中 0 ≤ λ ≤ 1，表示 a 和 b 之间的任意位置
```

翻译成人话：直线上算出来的值（右边），比曲线上算出来的值（左边），对于凸函数更大，对于凹函数更小。

#### 角度二：二阶导数判别法（代数定义）

如果你会求导，这是最快的判断方法：

| 二阶导数       | 函数类型       | 类比    |
| ---------- | ---------- | ----- |
| f''(x) > 0 | **凸函数**    | 碗 🥣  |
| f''(x) < 0 | **凹函数**    | 山 ⛰️  |
| f''(x) = 0 | 既凸又凹（线性函数） | 平面 📐 |

为什么二阶导数能判断凹凸？二阶导数是”斜率的变化率”：

- f''(x) > 0 意味着**斜率在不断增大**（越来越陡），你从左侧进入碗底再到右侧，坡度从负 → 0 → 为正，这就是碗形
- f''(x) < 0 意味着**斜率在不断减小**（越来越平），你爬山到顶再下坡，坡度从正 → 0 → 为负，这就是山形

#### 用 f(x) = x² 验证

```
f(x)  = x²
f'(x) = 2x     ← 一阶导数（斜率）
f''(x)= 2      ← 二阶导数恒为正数 → 凸函数 ✅
```

画出图像就是一个碗，完全吻合。

### 6.3 快速判断口诀

```
“碗” 能盛水 → 凸（Convex — 像碗一样凹进去的是凸）
“山” 会滑水 → 凹（Concave — 像山一样凸出来的是凹）
```

**一个常见的坑**：中文的”凸”和”凹”字面意思容易和英文反着理解：

- 中文”凸”字 → 中间鼓起来 → 但数学里的凸函数反而是**碗形**（中间凹下去）❓
- 记法：别管中文，记住 **Convex = 碗** 就行了

### 6.4 举例：几个常见函数

| 函数           | f''(x)    | 类型    | 图像      |
| ------------ | --------- | ----- | ------- |
| f(x) = x²    | 2 > 0     | **凸** | 🥣 碗    |
| f(x) = eˣ    | eˣ > 0    | **凸** | 🥣 左低右高 |
| f(x) = -x²   | -2 < 0    | **凹** | ⛰️ 山    |
| f(x) = ln(x) | -1/x² < 0 | **凹** | ⛰️ 越来越平 |
| f(x) = 2x+3  | 0         | 既凸又凹  | 📐 直线   |

### 6.5 为什么线性回归的损失函数是凸函数？

损失函数 `L = (wx + b - y)²` 是 `(预测值 - 真实值)²` 的形式。

它本质上是一个**开口朝上的抛物线**（二次函数），二阶导数为正常数：

```
∂²L/∂w² = 2x² > 0（通常情况下）
```

所以它是凸函数，只有一个全局最小值——这就是最小二乘法和梯度下降都能放心使用的根本原因。

### 6.6 凸函数在机器学习中的”无敌”特性

凸函数的两个最重要的性质，直接决定了机器学习中优化问题的难易程度：

> **性质 1：局部最小 = 全局最小**
> 
> 凸函数只有**一个**谷底。你找到的任何”看起来像谷底”的地方，就是真正的全局最低点。不会出现”你以为到了谷底，其实旁边还有个更深的坑”。

> **性质 2：梯度下降一定收敛**
> 
> 在凸函数上，梯度下降每一步都在减小损失值（只要学习率不设得太大），并且最终一定会到达最小值附近。

如果一个损失函数**不是**凸的（比如神经网络的损失函数），梯度下降就可能卡在局部最小值里出不来，这也是深度学习比线性回归更难训练的主要原因之一。

## 八、偏导数：当函数有多个"按钮"时

### 8.1 从普通导数到偏导数

普通导数面对的问题只有一个变量：

```
f(x) = x²
→ f'(x) = 2x  （变化率只取决于这一个 x）
```

但现实中的问题往往有**多个变量同时影响结果**。线性回归的损失函数就是典型：

```
L(w, b) = (1/2m) Σ (wxᵢ + b - yᵢ)²
         ↑   ↑
      两个参数同时影响损失值
```

这时普通的"导数"不够用了——**你到底在问谁对谁的变化率？**

> **偏导数**：固定其他所有变量不动，只让其中一个变量变化，看看函数值随之变化多快。

关键就在那个"偏"字——你只关注**一个**方向的变化，其他方向先不管。

### 8.2 用一个三维地形来建立直觉

把损失函数想象成一片山区的地形：

```
        ↑ 高度（损失值 L）
        |
        |    ⛰️  山峰
        |   /‾‾‾\
        |  /     \
        | /   ·   \    ← 你站在这里，想知道哪边是下山路
        |/_________\___→ 东西方向（参数 w）
        /          
       / 南北方向（参数 b）
```

你站在山上的某个点，想知道"往哪儿走能下山"：

- 问"**东西方向的坡度有多陡？**" → 这就是 **∂L/∂w**（固定 b，只看 w 方向的变化率）
- 问"**南北方向的坡度有多陡？**" → 这就是 **∂L/∂b**（固定 w，只看 b 方向的变化率）

### 8.3 计算偏导数的具体步骤

以 `L(w, b) = w² + b²` 这个最简单的二元函数为例：

**求 ∂L/∂w（对 w 的偏导）：**

```
步骤：把 b 当作常数，其他照常求导

L = w² + b²
     ↑     ↑
   对w求  把b²当常数，常数的导数为0

∂L/∂w = 2w + 0 = 2w
```

**求 ∂L/∂b（对 b 的偏导）：**

```
步骤：把 w 当作常数，其他照常求导

L = w² + b²
     ↑     ↑
  当常数  对b求导

∂L/∂b = 0 + 2b = 2b
```

用一张表对比就清楚了：

| 操作      | ∂L/∂w     | ∂L/∂b     |
| ------- | --------- | --------- |
| w² 怎么处理 | 正常求导 → 2w | 当作常数 → 0  |
| b² 怎么处理 | 当作常数 → 0  | 正常求导 → 2b |
| 结果      | 2w        | 2b        |

### 8.4 符号约定

```
普通导数：d/dx     （一个弯的 d）
偏导数：  ∂/∂x     （一个卷曲的 ∂，读作 "partial" 或 "round d"）
```

写成 ∂ 而不是 d，就是为了提醒你：**这个函数还有其他变量，我现在只对这一个变量求导**。

### 8.5 所有偏导数拼起来 = 梯度

把每个参数的偏导数写在一起，组成一个向量，就是**梯度**：

```
         ⎡ ∂L/∂w ⎤
∇L(w,b) = ⎢       ⎥
         ⎣ ∂L/∂b ⎦
```

> **梯度 = 所有偏导数组成的向量，指向函数上升最快的方向。**

回到山的类比：

- ∂L/∂w 告诉你东西方向有多陡
- ∂L/∂b 告诉你南北方向有多陡
- 把这两个信息**合并**，你就知道**最陡的上山路朝哪个方向**——这就是梯度
- 往梯度的**反方向**走，就是**最陡的下山路**——这就是梯度下降

### 8.6 回到线性回归

线性回归损失函数的偏导数（忽略前面的 1/2m 系数）：

```
L(w,b) = (wxᵢ + b - yᵢ)²

∂L/∂w = 2(wxᵢ + b - yᵢ) · xᵢ     ← 对 w 求偏导，b 视为常数
∂L/∂b = 2(wxᵢ + b - yᵢ) · 1      ← 对 b 求偏导，w 视为常数
```

偏导数告诉我们每条数据"拉"了参数多少，所有数据的偏导数加在一起，就是总体的调整方向和幅度。

### 8.7 一个扎心的对比

|      | 普通导数 f'(x)   | 偏导数 ∂f/∂x                         |
| ---- | ------------ | --------------------------------- |
| 适用场景 | 只有一个变量       | 有多个变量                             |
| 直觉   | 斜坡的坡度        | 山坡某个方向的坡度                         |
| 求法   | 正常求导         | 把其他变量都当常数，再正常求导                   |
| 符号   | d            | ∂                                 |
| 例子   | f(x)=x² → 2x | f(x,y)=x²+y² → ∂f/∂x=2x, ∂f/∂y=2y |

> **一句话总结偏导数**：
> 
> 普通导数问的是"函数值随这个唯一变量变化的快慢"；
> 
> 偏导数问的是"别人都别动，就让这一个变量动，函数值变化的快慢"。
